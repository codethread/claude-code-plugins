# MCP Server Development

Build high-quality Model Context Protocol (MCP) servers for AI agents.

## What

**Skill**: `mcp-builder` - 4-phase process (research, implement, review, evaluate)
**Reference Docs**: Best practices, Python/TypeScript patterns, evaluation guides
**Scripts**: Evaluation testing harness

## Why

- Design tools for AI agents, not just API wrappers
- Workflow-focused (complete tasks, respect token limits)
- Evaluation-driven development
- Supports Python (FastMCP) and TypeScript (MCP SDK)
- Agent-centric design principles

## How

```bash

```
