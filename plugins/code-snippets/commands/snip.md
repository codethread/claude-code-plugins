# Quick Snippet (Alias)

Shorthand for common snippet operations.

## What to do

`/snip` is a quick alias:

1. **With argument** - Quick insert:
   ```
   /snip react-component
   ```

2. **With save flag** - Quick save:
   ```
   /snip -s my-snippet
   ```

3. **With list flag** - Quick filter:
   ```
   /snip -l react
   ```

4. **No arguments** - Recent snippets

## Examples

```
User: /snip
You: Recent snippets:
1. react-component
2. error-handler
3. api-route

User: /snip error-handler
You: [Prompts for variables and inserts]
```
