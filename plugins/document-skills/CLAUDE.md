# Document Processing Skills

Collection of skills for working with common document formats.

## Available Document Skills

### PDF (`pdf/`)
PDF manipulation and form filling - see `pdf/CLAUDE.md` for details.

**Common tasks:**
- Extract text and tables from PDFs
- Create new PDFs with reportlab
- Merge, split, or rotate PDF documents
- Fill out PDF forms
- Process scanned PDFs with OCR

### XLSX (`xlsx/`)
Excel spreadsheet operations - see `xlsx/CLAUDE.md` for details.

**Common tasks:**
- Create spreadsheets with formulas and formatting
- Read and analyze spreadsheet data
- Modify existing spreadsheets while preserving formulas
- Data analysis and visualization

### PPTX (`pptx/`)
PowerPoint presentation creation - see `pptx/CLAUDE.md` for details.

**Common tasks:**
- Create new presentations from scratch or templates
- Modify existing presentation content
- Work with slide layouts and design
- Add comments or speaker notes

### DOCX (`docx/`)
Word document editing with tracked changes - see `docx/CLAUDE.md` for details.

**Common tasks:**
- Create new Word documents
- Edit existing documents with tracked changes
- Work with comments and formatting
- Process legal, academic, or business documents

## General Principles

All document skills emphasize:
- Preserving existing structure and formatting when editing
- Using appropriate libraries for each format
- Following format-specific best practices
- Testing output to ensure quality

## Choosing the Right Tool

- For **structured data**: Use XLSX
- For **presentations**: Use PPTX
- For **text documents with formatting**: Use DOCX
- For **print-ready documents**: Use PDF
